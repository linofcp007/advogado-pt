# Checklists — Índice

> Listas de verificação acionáveis (formato `- [ ]`) para o contexto de um negócio tech em Portugal. Cada ficheiro remete para a referência relevante; para valores e limiares, ver sempre `references/valores-2026.md`.

- [`checklist-rgpd.md`](checklist-rgpd.md) — Conformidade RGPD para um negócio tech: bases de licitude, política de privacidade, RAT (Art. 30.º), DPA (Art. 28.º), cookies, segurança, data breach (72h), direitos dos titulares (1 mês), transferências (SCC), DPIA e IA/decisões automatizadas (Art. 22.º).
- [`checklist-due-diligence-imovel.md`](checklist-due-diligence-imovel.md) — Verificação antes de comprar imóvel: documentos, ónus/encargos, urbanístico, condomínio, fiscal (IMT/IS), CPCV, escritura e registo imediato.
- [`checklist-constituicao-sociedade.md`](checklist-constituicao-sociedade.md) — Abrir uma Lda ou transitar de ENI para Lda: decisão fiscal, firma e CAE, pacto social, capital, constituição, registos, RCBE, contabilidade, conta bancária, transição de contratos e obrigações periódicas.
- [`checklist-revisao-contrato.md`](checklist-revisao-contrato.md) — Rever um contrato antes de assinar: partes, objeto, preço, prazos, PI, limitação de responsabilidade, confidencialidade, dados, rescisão, força maior, foro e cláusulas proibidas em CCG.
- [`checklist-predeploy-legal.md`](checklist-predeploy-legal.md) — Prontidão legal antes de lançar produto/feature: T&C, privacidade e cookies, consentimentos, DPA cloud, open-source, marcas (INPI), transparência de IA, consumo, segurança (CRA/NIS2) e seguros.
